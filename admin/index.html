打开前台

使用说明

1. 在「疾病条目」中可一次性编辑疾病信息、影像图片与说明
2. 修改后点击「保存全部疾病」写入 data/site-data.json
3. 前台会自动优先读取该 JSON 覆盖内置数据
4. 「影像图库」可按 type 批量维护（与疾病条目数据相同）

疾病条目

每条疾病包含：标题、分类、简介等文字信息，以及对应 type 键下的影像图片（file / caption / site）。点击「编辑」可在同一页面增减疾病与图片。

预览标题type区域分类图片操作

影像图库

按 type 键批量编辑图片。日常建议在「疾病条目」中编辑；此处适合对照多个 type 或批量维护。
