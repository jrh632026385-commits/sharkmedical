使用说明

1. 在「疾病条目」中可一次性编辑疾病信息、影像图片与说明
2. 修改后点击「保存全部疾病」写入 data/site-data.json
3. 前台会自动优先读取该 JSON 覆盖内置数据
4. 「影像图库」可按 type 批量维护（与疾病条目数据相同）

疾病条目

每条疾病包含：标题、分类、简介等文字信息，以及对应 type 键下的影像图片（file / caption / site）。点击「编辑」可在同一页面增减疾病与图片。

编辑疾病条目

学习内容（前台详情页）

保存后将在用户打开该疾病详情时展示。列表项每行一条，条数不限（处理原则/鉴别诊断/诊断陷阱/临床要点等有多少写多少）。带「标签|说明」的字段每行格式为 标签|说明（如 MRI|首选检查…）。

影像图片

图片挂在上方 type 键下。每张图可填写影像授权信息；留空则自动从 Wikimedia 拉取。

添加用户

创建后可登录前台或后台；普通用户仅可浏览，无内容编辑权限。
